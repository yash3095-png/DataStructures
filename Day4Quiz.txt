1.Does this compile? class A { protected void show(){} } class B extends A { void show(){} }
The correct answer is: Yes

2.Does this compile? class A { void test() {} } class B extends A { int test() { return 10; } }
The correct answer is: No

3.What is the output? class A { A show(){ System.out.println("A"); return this; } } class B extends A { B show(){ System.out.println("B"); return this; } } A obj = new B(); obj.show();

The correct answer is: B
